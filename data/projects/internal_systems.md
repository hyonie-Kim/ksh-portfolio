## 3. 핵심 개발 성과

### 요금제관리 시스템 신규 개발

![요금제관리 시스템](/images/projects/mng_price.png)

**개발 배경**  
매달 통신 요금제 정책 변경 시 반복적인 쿼리 작업을 웹 기반 관리자 페이지로 자동화

**주요 기능**
- 정책 조회 및 일괄 변경
- 외부 연동 API 대응값 관리
- 관련 내부 데이터 일괄 업데이트
- 중복 등록 방지 및 데이터 무결성 검증

---

### 외부 고객사 입금 API 신규 개발

![입금 API 테스트](/images/projects/deposit_api_test.png)

**개발 배경**  
외부 고객사의 입금 확인에 따른 계약 상태 변경 및 업무큐 처리를 위한 REST API 개발

**API 명세 (일반화)**
- **Endpoint**: `POST /api/v1/payment-status`
- **목적**: 입금 확인에 따른 계약 상태 변경 및 업무큐 처리
